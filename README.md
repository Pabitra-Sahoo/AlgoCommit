## 🚀 Core Features

- **📂 Structured GitHub Sync**: Automatically sync all your solved problems from **LeetCode**, **GeeksForGeeks**, and **Codeforces** to your GitHub repository. Every solution is organized into clean, platform-specific folders with dedicated READMEs and difficulty labels.
- **🚀 Atomic Sync Engine**: Guarantees zero data loss. Your solution, problem description, and stats are pushed in one single, clean transaction.
- **📊 30-Day Activity Heatmap**: Visualize your consistency with a GitHub-style activity grid right in your browser popup.
- **🛡️ Engineering Ranks**: Gamify your learning. Level up from **Learner** to **Tech Lead** as you solve more problems.
- **🔥 Persistence Tracking**: Stay motivated with a robust daily streak system and automated reminders so you never miss a day.
- **🔗 Quick Access**: Use the "View Repo" button to jump straight to your synced code whenever you need it.

---

## ⚙️ How It Works

1. **Solve**: Solve a problem on LeetCode, GeeksForGeeks, or Codeforces.
2. **Detect**: AlgoCommit automatically detects the "Accepted" status directly in your browser.
3. **Commit**: Your solution, along with problem details, is instantly pushed to your linked GitHub repository in a single atomic commit.

---

## 🔒 Your Code, Your Privacy

**AlgoCommit puts you in control.** Unlike other tools, we don't store your code on our servers.
- **100% Ownership**: Every solution, every stat, and every byte of data is stored directly in **your own GitHub repository**.
- **Privacy First**: You own your progress history forever. Even if you uninstall the extension, your code remains safe in your repo.
- **No Middlemen**: It's a direct, secure connection between your browser and GitHub.

## 🛠️ Built With

AlgoCommit is built using modern web technologies for maximum performance and security:

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Modern Glassmorphism UI)
- **API**: [GitHub REST & Git Data API](https://docs.github.com/en/rest)
- **Runtime**: Chrome Extension API (Manifest V3)

## 📦 Installation (Developer Mode)

1. Clone this repository to your local machine.
2. Run `npm install` to install dependencies.
3. Run `npm run build` to generate the production build.
4. Open Chrome-based browser and go to `chrome://extensions`.
5. Enable **Developer Mode** (top right).
6. Click **Load unpacked** and select the `dist` folder.
