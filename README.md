# AlgoCommit: Auto-Sync DSA Solutions

AlgoCommit is a powerful browser extension designed for developers and competitive programmers. It automatically syncs your solved problems from **LeetCode**, **GeeksForGeeks**, and **Codeforces** directly to your GitHub repository in real-time.

![AlgoCommit Logo](public/Myicon.png)

## ✨ Features

- **🚀 Atomic Sync Engine**: Guarantees your code, problem descriptions, and stats are committed as a single transaction—no partial syncs or race conditions.
- **📊 30-Day Activity Heatmap**: Visualize your consistency with a beautiful, GitHub-style 30-day activity grid.
- **🛡️ Engineering Ranks**: Earn ranks from **Learner** to **Tech Lead** as you solve more problems. Features custom SVG badges and rank-up animations.
- **🔥 Persistence Tracking**: Robust streak system with daily reminders and milestone celebrations (10, 25, 50, 100+ problems).
- **⏱️ Peace-of-Mind Sync**: "Last Synced" timestamps and tooltips ensure you always know your progress is safe.
- **🔗 Deep GitHub Integration**: One-click "View Repo" button to jump straight to your synced code.

## 🛠️ Supported Platforms

- **LeetCode**: Full support for all problem types and languages.
- **GeeksForGeeks**: Automatic capture of successful submissions.
- **Codeforces**: Real-time monitoring of the status page for accepted solutions.

## 📦 Installation (Developer Mode)

1. Clone this repository.
2. Run `npm install` and then `npm run build`.
3. Open Chrome/Edge and navigate to `chrome://extensions`.
4. Enable **Developer Mode**.
5. Click **Load unpacked** and select the `dist` folder.
